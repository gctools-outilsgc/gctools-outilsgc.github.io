About Code GCTools

What is Code GCTools?

  • Code GCTools is an event in which GC employees will collaborate to improve GCTools by crowd sourcing resources and skills from computer programmers, designers, project managers, writers/editors, artists, team builders - or anyone who has an innovative idea they would like to see integrated into the platform.
  • The goal of the Code GCTools is to allow members of the GC to build better GCTools. Employees from every area of the GC will be invited not only to submit their vision of improving the GCTools experience but will also be encouraged to provide the necessary coding and design for enhancements.
  • To further improve the platforms, the Service and GC2.0 Policy and Community Enablement (SGCPCE) Division at TBS is organizing a codefest event in April 2017.
  • In advance of the codefest, and, in order to build capacity, the GC2.0 Tools team will host virtual workshops.
